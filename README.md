# WABOX
Simple WhatsApp bot to create sticker from image, video or GIF.

## How to use
```sh
git clone https://github.com/d0tf/wabox
cd wabox
npm install
npm run compile
npm start
```

#### Run Inside Docker
```sh
git clone https://github.com/d0tf/wabox
cd wabox
docker compose up -d
docker logs wabox # to see qrcode
```
